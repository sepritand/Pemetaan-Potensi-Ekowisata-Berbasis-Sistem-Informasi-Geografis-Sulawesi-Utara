Latar Belakang

Website ini merupakan hasil dari proyek tugas berjudul "Pemetaan Potensi Ekowisata Berbasis SIG di Sulawesi Utara". Proyek ini bertujuan untuk memetakan dan menyajikan informasi lokasi-lokasi ekowisata di Provinsi Sulawesi Utara secara interaktif menggunakan teknologi Sistem Informasi Geografis (SIG). Melalui peta digital yang tersedia, diharapkan masyarakat umum maupun pemerintah daerah dapat dengan mudah mengakses dan memanfaatkan data ekowisata ini untuk keperluan pengembangan dan promosi pariwisata berkelanjutan.

Teknologi yang Digunakan

  • HTML, CSS, dan JavaScript
  • Leaflet.js untuk tampilan peta
  • Data lokasi dalam format JSON
  • Visualisasi menggunakan XAMPP (Apache)
